# IB DP Geography SL - テーマ4：グローバルな変化における人間と自然環境のシステム

## 1. 生物圏

### 1.1 生物多様性と変化

生物多様性：遺伝的多様性、種の多様性、生態系の多様性を含む。

重要性：
- 生態系の安定性：例）サンゴ礁における種の多様性が生態系の回復力を高める
- 食料安全保障：例）遺伝的に多様な作物が病害虫に強い
- 医薬品開発：例）熱帯雨林の植物からの新薬開発

脅威：
1. 生息地の破壊：例）アマゾンの森林伐採によるジャガーの生息地減少
2. 気候変動：例）北極海の氷減少によるホッキョクグマへの影響
3. 外来種：例）オーストラリアのサトウキビ畑に導入されたオオヒキガエルの生態系破壊
4. 乱獲：例）大西洋のタラの個体数激減

### 1.2 生態系と生物群系

主要な生物群系：
1. 熱帯雨林：例）アマゾン - 多層構造、高い生物多様性
2. 温帯林：例）ドイツの黒い森 - 落葉樹と常緑樹の混在
3. 針葉樹林：例）シベリアのタイガ - 寒冷適応の常緑樹
4. 草原：例）セレンゲティ - 季節的な降雨、大型草食動物
5. 砂漠：例）サハラ - 極度の乾燥、耐乾性植物
6. ツンドラ：例）アラスカ北部 - 永久凍土、短い生育期間

生態系サービス：
- 供給：例）アマゾンの薬用植物
- 調整：例）マングローブによる海岸保護
- 文化的：例）日本の富士山の審美的・宗教的価値
- 支持：例）熱帯雨林の光合成による酸素供給

### 1.3 生物地理学

主要概念：
1. 島嶼生物地理学：例）ガラパゴス諸島のフィンチの進化
2. 種の分散：例）ココヤシの海流による長距離分散
3. 絶滅と再植民：例）カリフォルニアコンドルの絶滅危機と保護活動

生物地理学的パターン：
- 緯度勾配：例）赤道から極への種数減少
- 高度勾配：例）キリマンジャロ山の植生帯の変化

## 2. 大気と気候

### 2.1 大気循環と気候パターン

大気大循環：
1. ハドレー循環：例）赤道付近の上昇気流がアマゾンの多雨をもたらす
2. フェレル循環：例）日本付近の偏西風による気候への影響
3. 極循環：例）北極振動による北半球中緯度の寒波

気候区分（ケッペン）：
- A（熱帯）：例）シンガポール - 年中高温多湿
- B（乾燥）：例）エジプト・カイロ - 極度の乾燥
- C（温帯）：例）フランス・パリ - 四季の変化
- D（寒冷）：例）ロシア・モスクワ - 寒冷な冬
- E（極地）：例）南極・マクマード基地 - 極寒

エルニーニョ南方振動（ENSO）：
- 影響：例）ペルーの豪雨、オーストラリアの干ばつ

### 2.2 気候変動

原因：
- 自然要因：例）ピナトゥボ火山噴火（1991年）による一時的な寒冷化
- 人為的要因：例）化石燃料燃焼によるCO2増加

影響：
1. 海面上昇：例）ツバルの海岸侵食と塩水侵入
2. 極端気象：例）2003年ヨーロッパ熱波、2019-20年オーストラリア森林火災
3. 生態系変化：例）北極圏の植生変化（ツンドラの灌木化）

対策：
- 緩和策：例）ドイツのエネルギー転換政策（再生可能エネルギーの促進）
- 適応策：例）オランダの「水と共に生きる」プログラム（洪水対策）

### 2.3 気象ハザード

主な気象ハザード：
1. 熱帯低気圧：例）2005年ハリケーン・カトリーナ（米国）
2. 洪水：例）2011年タイ大洪水
3. 干ばつ：例）2011-2017年カリフォルニア干ばつ
4. 熱波：例）2019年インド熱波

影響：
- 社会経済的：例）農業生産の減少、インフラ破壊
- 生態系：例）森林火災による生息地喪失

管理戦略：
1. 早期警報システム：例）日本の緊急地震速報
2. 土地利用計画：例）ニューオーリンズの湿地再生プロジェクト
3. コミュニティの準備：例）バングラデシュのサイクロンシェルター建設

## 3. 水文学と水資源

### 3.1 水循環

主要プロセス：
1. 蒸発：例）太平洋からの水蒸気が日本の梅雨前線の水源となる
2. 降水：例）ヒマラヤ山脈における地形性降雨
3. 浸透：例）カルスト地形における地下水涵養
4. 流出：例）アマゾン川の年間流量変動

人間活動の影響：
- 土地利用変化：例）インドネシアのパーム油プランテーションによる森林減少と流出量増加
- 都市化：例）東京の不浸透面積増加による洪水リスク上昇
- ダム建設：例）ナイル川アスワンハイダムによる河川流量の調節

### 3.2 河川システム

河川地形：
1. 上流：例）スイスアルプスの急流と滝
2. 中流：例）ライン川中流の蛇行と氾濫原
3. 下流：例）ガンジス川デルタの三角州と分流

人間の介入：
- 河川改修：例）オランダのライン川デルタプロジェクト（洪水防止と航行改善）
- ダム：例）中国の三峡ダム（発電と洪水制御）
- 灌漑：例）カリフォルニア中央峡谷の大規模灌漑システム

### 3.3 水資源管理

淡水の可用性と分布：
- 地域差：例）カナダ（豊富）vs サウジアラビア（乏しい）
- 季節変動：例）インドのモンスーン期と乾季の水利用の違い

水ストレス：
- 原因：人口増加、気候変動、汚染
- 影響：例）アラル海の縮小による地域生態系と経済への影響

持続可能な水管理：
- 統合水資源管理（IWRM）：例）オーストラリアのマレー・ダーリング川流域管理
- 水の再利用：例）シンガポールのNEWater（下水再生水）プロジェクト

## 4. 沿岸環境

### 4.1 沿岸プロセスと地形

波浪、潮汐、海流の影響：
- 波浪浸食：例）イギリス・ドーセット海岸の海食崖
- 潮汐：例）カナダ・ファンディ湾の世界最大潮差
- 海流：例）日本の黒潮による気候と漁業への影響

主要な沿岸地形：
1. 海食崖：例）オーストラリアの十二使徒岩
2. 砂浜：例）ブラジル・コパカバーナビーチ
3. 三角州：例）ミシシッピ川デルタ
4. マングローブ林：例）スンダルバンス（インド・バングラデシュ）

### 4.2 珊瑚礁生態系

分布と形成条件：
- 分布：例）グレートバリアリーフ（オーストラリア）
- 条件：水温20-30℃、塩分濃度32-35‰、十分な日光

脅威：
1. 海洋酸性化：例）カリブ海におけるサンゴの石灰化率低下
2. 海水温上昇：例）1998年のエルニーニョによる大規模白化現象
3. 乱獲：例）インドネシアにおけるダイナマイト漁法の影響

保全戦略：
- 海洋保護区：例）パラオの国家海洋保護区（商業漁業全面禁止）
- 持続可能な観光：例）モルディブのエコリゾート開発

### 4.3 沿岸管理

海面上昇の影響と適応策：
- 影響：例）モルディブの島々の水没リスク
- 適応策：例）オランダの「すてきな適応（Room for the River）」プログラム

沿岸管理アプローチ：
1. ハードエンジニアリング：
   例）日本の防波堤建設（東日本大震災後の復興事業）
2. ソフトエンジニアリング：
   例）デンマークの砂浜養浜プロジェクト

統合沿岸域管理（ICZM）：
- 原則：例）EU沿岸域統合管理勧告（生態系アプローチ、長期的視点）
- 実践：例）スペイン・カタルーニャ地方のICZM計画（観光と環境保護の両立）

## 5. 地質学的プロセスとリスク

### 5.1 プレートテクトニクス

主要なプレート境界：
1. 発散境界：例）東アフリカ大地溝帯（新しい海洋の形成過程）
2. 収束境界：例）南米アンデス山脈（海洋プレートと大陸プレートの衝突）
3. トランスフォーム断層：例）カリフォルニアのサンアンドレアス断層

関連する地形：
- 中央海嶺：例）大西洋中央海嶺
- 海溝：例）マリアナ海溝（世界最深部）
- 造山帯：例）ヒマラヤ山脈（インド・ユーラシアプレート衝突）

### 5.2 火山活動

火山のタイプ：
1. 盾状火山：例）ハワイのマウナロア（玄武岩質溶岩）
2. 成層火山：例）日本の富士山（交互に堆積した溶岩と火山灰）
3. カルデラ：例）インドネシア・トバ湖（超巨大噴火の跡）

火山ハザード：
- 溶岩流：例）2018年ハワイ・キラウエア噴火
- 火山灰：例）2010年アイスランド・エイヤフィヤトラヨークトル噴火による欧州航空混乱
- 火砕流：例）1902年マルティニーク島ムон・プレー噴火による市街地壊滅

火山リスク管理：
- モニタリング：例）イタリア・ヴェスヴィオ火山観測所
- ハザードマッピング：例）米国地質調査所（USGS）の火山ハザードマップ
- 避難計画：例）フィリピン・ピナツボ火山1991年噴火時の大規模避難成功

### 5.3 地震

原因：
- 断層運動：例）2008年中国四川大地震（竜門山断層）
- プレート境界：例）2011年東日本大震災（日本海溝）

地震の測定：
- マグニチュード：例）リヒタースケール、モーメントマグニチュード
- インテンシティ：例）気象庁震度階級、メルカリ震度階級

地震リスク管理：
1. 建築基準：例）日本の耐震設計基準強化
2. 早期警報システム：例）メキシコの地震早期警報システム（SASMEX）
3. コミュニティの準備：例）ニュージーランドの「シェイクアウト」訓練
